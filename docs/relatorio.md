# Principais Comandos Git Utilizados
 -	git checkout -b <branch> → Criar nova branch para uma tarefa específica.
 -	git add <arquivo> → Adicionar arquivos modificados para commit.
 -	git commit -m "<mensagem>" → Criar commit com mensagem clara.
 -	git push origin <branch> → Enviar branch para GitHub.
 -	git pull origin main → Atualizar a branch local com mudanças da main.
 -	git branch -D <branch> → Deletar branch local obsoleta.
